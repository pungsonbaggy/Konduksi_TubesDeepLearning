# Face Recognition for Student Attendance

Aplikasi berbasis Deep Learning untuk mengenali wajah mahasiswa Institut Teknologi Sumatera.

## Fitur
- Deteksi wajah menggunakan Vision Transformer (ViT)
- Presensi otomatis dengan confidence score
- Antarmuka web yang responsif

## Teknologi
- Model: Vision Transformer (ViT)
- Framework: TensorFlow 2.20.0
- Frontend: Streamlit
- Deployment: Hugging Face Spaces
